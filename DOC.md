## 1. Deskripsi Aplikasi
Aplikasi ini adalah **Single Page Application (SPA)** yang dibangun menggunakan **React JS** dan **Vite**, serta didesain menggunakan **Tailwind CSS**. Aplikasi ini berfungsi untuk menampilkan Curriculum Vitae (CV) mahasiswa secara interaktif, modern, dan responsif.

**Fitur Utama:**
* **Struktur Modular:** Terbagi menjadi 3 bagian utama yaitu **Profile**, **Skills**, dan **Experiences**.
* **Dynamic Data:** Seluruh data diri (Bio, Pengalaman, Skill) diambil dari file `src/dataMahasiswa.json`, sehingga tidak *hardcode* di dalam komponen.
* **Modern UI/UX:** Menggunakan fitur **Dark Mode**, **Sticky Navbar** dengan efek *glassmorphism*, dan animasi transisi yang halus.
* **Asset Management:** Menggunakan logo teknologi asli dan icon dari `lucide-react` untuk tampilan yang lebih profesional.

## 2. Cara Menjalankan (Local)
Berikut adalah langkah-langkah untuk menjalankan aplikasi ini di komputer lokal:

1.  **Pastikan Node.js sudah terinstall.**
2.  **Buka terminal** di folder proyek ini.
3.  **Install dependencies** (pustaka yang dibutuhkan):
    ```bash
    npm install
    ```
4.  **Jalankan server development:**
    ```bash
    npm run dev
    ```
5.  **Buka browser** dan akses URL yang muncul (biasanya `http://localhost:5173`).
